# ♣️ClubOps AI - *"_One Prompt = Complete Event Management_"*

## [Live Demo](https://clubops-ai-vercel-o7dyd34ei-kauongits-projects.vercel.app/) ## [Backend API](https://clubops-ai.onrender.com)

ClubOps AI is an end-to-end generative AI platform designed to help college clubs
plan, execute, and document events using a single prompt.

## Problem
Student clubs rely on multiple tools for posters, emails, certificates,
attendance, and reports. This leads to inefficiency, inconsistency, and
last-minute chaos.

## Solution
ClubOps AI automates the entire event lifecycle by generating all required
assets from one prompt, enabling non-technical students to manage professional
events easily.

ClubOps AI takes basic event details as input and orchestrates multiple
generative AI components to produce ready-to-use event assets in a structured
workflow.

## Impact
ClubOps AI reduces time spent on event operations, improves consistency across
club communications, and enables non-technical students to run professional
events with minimal effort. This helps clubs focus more on creativity,
participation, and community building rather than logistics.

## Features
- AI Poster Generation
- Social Media Caption & Content Generation
- Email & Invitation Generator
- Certificate Generator
- Attendance Sheet Generation
- Post-event Summary & Report

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express
- AI: Open-source LLMs (LLaMA/Mistral), Stable Diffusion
- Database: MongoDB
- Hardware Alignment: AMD Ryzen, EPYC, Instinct GPUs (ROCm)
